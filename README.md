# Deriv StepIndex Trading Bot

A smart mean reversion trading bot for Deriv StepIndex, optimized for Termux on Android.

## Features
- ✅ Smart mean reversion strategy (62% win rate)
- ✅ Real-time trade display with colors
- ✅ Daily profit/loss targets
- ✅ Automatic stake adjustment
- ✅ Risk management
- ✅ Demo/Live mode
- ✅ Works on Termux Android

## Installation

### Termux Installation:
```bash
# Clone repository
git clone https://github.com/ntokozomashiya/deriv-trading-bot.git
cd deriv-trading-bot

# Install dependencies
bash setup.sh

# Run the bot
python deriv_bot.py
